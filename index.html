Fair bandit is a provably-fair online gambling built on Ethereum blockchain. It is based on Ethereum smart contract. Strictly speaking it is exactly an Ethereum smart contract and this page is only a graphical user interface to it. Actually you can play this game bypassing this GUI just calling appropriate function in contract, but we don't recommend to do it, if you sre not experienced Ethereum user. You just should understand that if you make a stake, your game will be played and you will get you gain anyway.

So, how does this smart contrac work, and is it fair? First of all, once smart contract is deployed to blockchain, it can't be changed, and you can be sure that the rules stay constant. Besides, the result of the game is really random, and wa don't have any influence to the random supplier(http://random.org).

Now let's see what is going on, when you place the stake. As soon as you confirm transaction to this contract, it calls oraclize((http://www.oraclize.it/) smart contract. With the help of oraclize we get three random numbers from random.org. Each obtained number means a cell in a corresponding barrel. Fist number corresponds to first barrel and so on. On the right you can see how each cell of each barrel corresponds to icon from pay table. For example, we get 6, 1, 9. It means that we have three double barrels, and the stake is multiplied to 12. Smart contract will automatically count your gain and pay it to your wallet. That's it!

Below you can check the source code of the contract and make sure that it works exactly the same way as it was mentioned before. You can also navigate to the etherscan page of the contract and read it there.
